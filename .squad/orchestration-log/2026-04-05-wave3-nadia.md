## Architecture

- **Input narrowing:** All validators accept `unknown` with runtime checks
- **Structured results:** Every validator returns `ValidationResult<T>` — no exceptions, no void+throw
- **Security:** `sanitizeFilename` prevents path traversal (`../../`), `sanitizeString` strips control chars
- **Type safety:** Generic `ValidationResult<T>` enables extensible error handling
- **Immutability:** `readonly` annotations throughout
